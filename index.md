## 第一堂课笔记：算法 (Algorithm) vs. 模型 (Model)

### 核心概念解析

在生物信息学，尤其是机器学习应用中，**算法**和**模型**是两个紧密相关但本质不同的概念 [citation:5]。

*   **🧠 算法 (Algorithm)**：
    *   **定义**：算法是一套**解决问题的步骤和指令**，它是一个“过程”或“方法”。可以把它想象成一份**菜谱**，详细说明了从准备食材到烹饪完成的每一个具体步骤。
    *   **作用**：用于从数据中学习模式或进行预测。它是一个通用的、未在特定数据上运行的逻辑。
    *   **例子**：**“决策树算法”** 描述了如何根据数据特征（如基因表达量）来递归地划分数据，形成一个树状结构。这个算法本身不包含任何具体的生物学知识。

*   **🤖 模型**：
    *   **定义**：模型是**算法在特定数据集上运行后产生的具体“产物”**。它包含了从数据中学习到的“知识”或“规则”。可以把它想象成按照那份**菜谱（算法）**，用你手里**具体的食材（数据）** 烹饪出的**一道菜（模型）**。
    *   **作用**：用于对新的、未知的数据进行预测。
    *   **例子**：用“决策树算法”分析了一个包含1000个癌症样本和1000个正常样本的基因表达数据集后，算法生成的那个具体的、带有分裂规则的树，就是我们训练得到的**“癌症预测模型”**。

### 一张图看懂区别

| 特征 | 算法 (Algorithm) | 模型 (Model) |
| :--- | :--- | :--- |
| **本质** | 方法、过程、步骤 | 产物、结果、知识集合 |
| **状态** | 静态的、通用的 | 动态的、由数据和算法共同决定 |
| **依赖** | 独立于数据 | 完全依赖于训练它的数据 |
| **类比** | 菜谱、配方 | 按照菜谱做出的那道菜 |
| **目标** | 如何从数据中学习 | 学到的结果是什么，能做什么预测 |

### 小结
简单来说，**我们用一个“算法”在数据上进行“训练”，最终得到的是一个“模型”**。评估一个算法看它的效率和复杂度，而评估一个模型则看它的预测准确率、泛化能力等。

## 本学期生物信息学学习计划

为了让学习更有条理和效率，我为本学期制定了如下“务实”的学习计划，专注于核心技能的掌握和实际应用。

### 🎯 总体目标
*   **基础**：掌握 Linux 环境下的基本操作和 R/Python 语言的核心数据分析包。
*   **核心**：能够独立完成一个标准的转录组差异表达分析流程。
*   **应用**：理解机器学习在生物信息学中的基础应用，并能复现一篇简单生信论文的结果。

### 📅 阶段性学习安排

**第一阶段：筑基（第1-5周）**
- [x] **第1周**：完成 GitHub 环境配置，掌握 Git 进行代码版本管理。**（就是现在！）**
- [ ] **第2-3周**：系统学习 **R语言** (tidyverse 数据清洗、ggplot2 数据可视化) 和 **Linux 常用命令** (ls, cd, grep, awk, sed) [citation:6]。
- [ ] **第4-5周**：学习 **Python 基础** (NumPy, Pandas) 用于处理大型生物数据文件 (如 FASTQ, GTF 格式)。

**第二阶段：核心流程实战（第6-12周）**
- [ ] **第6-7周**：学习二代测序数据基础，掌握 **FASTQ** 文件格式和质量评估 (FastQC) [citation:6]。
- [ ] **第8-10周**：**转录组分析流程实战**：
    1.  序列比对 (Hisat2, STAR) -> 得到 SAM/BAM 文件。
    2.  定量 (featureCounts) -> 得到 Counts 矩阵。
    3.  **差异表达分析**：使用 R 包的 `DESeq2` 或 `limma` 寻找差异基因 [citation:6]。
- [ ] **第11-12周**：**结果可视化与功能解读**：
    1.  绘制火山图、热图。
    2.  **功能富集分析**：学习使用 `clusterProfiler` 进行 GO 和 KEGG 富集分析，解释生物学意义 [citation:6]。

**第三阶段：拓展与应用（第13-18周）**
- [ ] **第13-14周**：**机器学习入门**：
    1.  学习使用 `scikit-learn` (Python) 或 `caret` (R) 进行简单的分类 (SVM, Random Forest)。
    2.  理解 Lasso 回归在特征选择（筛选关键基因）中的应用 [citation:6]。
- [ ] **第15-16周**：**生存分析与模型评估**：
    1.  学习如何使用生存数据 (生存时间、状态)，进行单因素/多因素 Cox 回归分析。
    2.  绘制**生存曲线 (Kaplan-Meier)** 和 **ROC 曲线**，评估基因或模型的预测能力 [citation:6]。
- [ ] **第17-18周**：**综合实践**：
    1.  尝试完整复现一篇简单的、基于公共数据 (TCGA/GEO) 的预后模型类论文的分析流程。
    2.  整理代码和笔记到 GitHub 仓库。
